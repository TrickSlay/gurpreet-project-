Course Outline

Empathize and Define

Understand user needs with tech-driven research tools.

Ideate and Prototype

Generate ideas using AI and build digital prototypes.

Test and Iterate

Refine designs with data analytics and feedback loops.

Syllabus

  • Module 1: Empathize (Weeks 1-2) – User research with digital surveys, observation via apps, empathy mapping on virtual boards.
  • Module 2: Define (Weeks 3-4) – Problem framing using data visualization, point-of-view statements with NLP tools, user personas via AI.
  • Module 3: Ideate (Weeks 5-6) – Brainstorming with collaborative platforms, creative thinking tools like mind-mapping software, idea selection via voting apps.
  • Module 4: Prototype (Weeks 7-8) – Low-fidelity prototyping with Figma/Miro, wireframing for apps/websites, rapid experimentation with code snippets.
  • Module 5: Test (Weeks 9-10) – User testing via remote tools, feedback loops with analytics, iteration cycles using version control.
  • Final Project (Weeks 11-12) – Apply Design Thinking to a tech innovation challenge, e.g., app design.

Evaluation Criteria

Assessments include quizzes (30%), assignments (40%), and final project (30%). Pass with 70% or higher.
